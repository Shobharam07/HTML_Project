Nested List in HTML

A nested list in HTML is a list that contains other lists within its list items. This creates a hierarchical structure, where each sublist is indented to visually represent its relationship to the parent list item.

  1. Nested Unordered List in HTML

    2. An unordered list in HTML is a collection of items represented by bullet points, providing a flexible way to display information without a specific order.

    Lists can be nested (list inside list):

    Syntax:

    Example 1 
    <!DOCTYPE html>
<html>
<body>
<h2>A Nested List</h2>
<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>
</body>
</html>

    type="a" This will list the items in lowercase letters.
    type="I" his will list the items with uppercase Roman numbers.

Table in HTML

An HTML table is a way to organize and display data in a row-and-column format on a webpage. It is created using the <table> element in HTML. Tables are useful for showing structured information like schedules, price lists, or comparisons.

Basic Structure of an HTML Table:

  • <table> →Defines the table.
  • <tr>(Table Row) →Defines a row in the table.
  • <th> (Table Header)→Defines a header cell (bold and centered by default).
  • <td> (Table Data)→Defines a normal data cell.
  • <tfoot>→The <tfoot> element in HTML is used to group footer content in a table.
  • <caption>→The <caption> element is used to add a title or description to a table.
  • colspan:Used to merge multiple columns into one cell
  • rowspan:Used to merge multiple rows into one cell.
  • border:This attribute is used to define the thickness of the border (the lines) around the table and its cells.
Example 1 
<!DOCTYPE html>
<html>
<head>
    <title>HTML Table Example</title>
</head>
<body>
    <h2>Student Information</h2>
    <table border="1">
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Course</th>
        </tr>
        <tr>
            <td>Rahul</td>
            <td>20</td>
            <td>BCA</td>
        </tr>
        <tr>
            <td>Priya</td>
            <td>22</td>
            <td>MCA</td>
        </tr>
    </table>
</body>
</html>

Output:

Name Age Course
Rahul 20 BCA
Priya 22 MCA


HTML Table with Colspan

HTML Table with Colspan allows you to merge or combine adjacent table cells horizontally, creating a single, wider cell that spans across multiple columns.

Note: The colspan is defined as the element.

Example of colspan 
<!DOCTYPE html>
<html>
<head>
    <title>colspan Table Example</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Name</th>
            <th colspan="2">Subjects</th>
        </tr>
        <tr>
            <td>Rahul</td>
            <td>Math</td>
            <td>Science</td>
        </tr>
        <tr>
            <td>Priya</td>
            <td>English</td>
            <td>Computer</td>
        </tr>
    </table>
</body>
</html>

output:

Name Subjects
Rahul Math Science
Priya English Computer


HTML Table with Rowspan

The HTML attribute rowspan determines how many rows a specific cell in a table should cover. When a cell spans multiple rows, it occupies the space of those rows within the table.

Example of rowspan 
<!DOCTYPE html>
<html>
<head>
    <title>Rowspan Table Example</title>
</head>
<body>
    <table border="1">
        <tr>
            <th rowspan="2">Name</th>
            <th>Subject</th>
            <th>Marks</th>
        </tr>
        <tr>
            <td>Math</td>
            <td>90</td>
        </tr>
        <tr>
            <td>Rahul</td>
            <td>Science</td>
            <td>85</td>
        </tr>
    </table>
</body>
</html>

Output:

Name Subject Marks
Math 90
Rahul Science 85
