Media Tag

In HTML, "media tags" refer to the elements used to embed and control various types of multimedia content within a web page. These tags allow you to include images, audio, and video, enhancing the user experience beyond plain text.

Media Tags Description
<audio> An inline element is used to embed sound files into a web page.
<video> Used to embed video files into a web page.
<source> Used to attach multimedia files like audio, video, and pictures.
<embed> Used for embedding external applications, generally multimedia content like audio or video, into an HTML document.
<track> Specifies text tracks for media components, specifically for audio and video elements..

Advantage of Media Tag:

  • Plugins are no longer required
  • Speed - anything naturally integrated into a browser will be rendered and executed in a faster fashion than imported third-party
  • Native (built-in) controls are provided by the browser.
  • Accessibilities (keyboard, mouse) are built-in automatically

<audio> Tag:

The <audio> tag in HTML5 is used to embed audio content on a webpage. It allows you to play audio files like MP3, OGG, or WAV directly in the browser. The <audio> element provides attributes for controlling playback, such as play, pause, and volume.

Syntax:

            <audio>
                <source src="sample.mp3" type="audio/mpeg">
            </audio>

Attributes of <audio> Tag

  • controls →shows play, pause, volume buttons.
  • autoplay →plays automatically (may be blocked by browsers).
  • loop →repeats the audio automatically.
  • muted →starts audio in muted state.


<audio> tag Example 
<!DOCTYPE html>
<html>
  <head>
  </head> 
  <body>
    <p>Audio Sample</p>
    <audio>
      <source src= "/media/HTML_Day05_wavy.mp3"
        type="audio/mpeg">
    </audio>
  </body>
</html>

<video> Tag:

The <video> element in HTML is used to show video content on web pages. It supports various video formats, including MP4, WebM, and Ogg. It is introduced in HTML5.

Syntax:

<video src="" controls> </video>
or
<video controls="controls"> <source src="video_filename" type="video_type"> </video>


Attributes of <video> Tag

  • controls → adds play/pause/volume controls.
  • autoplay → starts video automatically.
  • loop → repeats video
  • muted → starts video muted.
  • poster="image.jpg" → shows an image before the video plays.
  • width & height → set video size.

<video> tag Example 
<!DOCTYPE html>
<html>
  <head>
  </head> 
  <body>
    <p>Video Tag Sample</p> 
    <video>
      <source src= "/media/rohit.mp4"
        type="video/mp4">  
    </video>
  </body>
</html>
